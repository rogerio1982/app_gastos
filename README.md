# ROBÔ TELEGRAM 

Bot teelgram para gestão de gastos.

## Pré-requisitos

- Python 3.10+
- pip

## Instalação

```bash
pip install -r requirements.txt
```

## Executar

```bash
python app.py
```

## Docker

```bash
docker build -t app_gastos .
docker run --rm -p 8000:8000 app_gastos
```

## Estrutura

- app.py
- requirements.txt
- Dockerfile
